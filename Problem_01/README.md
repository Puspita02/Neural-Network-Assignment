##Problem
-Pneumonia detection using chest X-ray images

##Approach
-Used CNN for image classification
-CNN automatically extracts features like edges, textures and pattern from image
-Applied data augmentation to improve generalization

##Methodology
-Preprocess image using ImageDataGenerator
-Rescaling pixel values from 0-255 to 0-1
-Data augmentation
-Convo2D layers
-MaxPooling layers
-Flatten layers
-Activation function ReLU for non-linearity
-Activation function Sigmoid for output layer
-Optimizer-Adam
-Epoch-10

##Findings
-Accuracy-0.86
